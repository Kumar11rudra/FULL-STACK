# Experiment 1.1.2

## Title

Draft Management System

## Aim

To implement a draft management system that allows users to save, retrieve, edit, and delete post drafts within the frontend.

## Objectives

- Manage draft data using React state.
- Implement CRUD operations.
- Understand asynchronous workflows.
- Persist data using localStorage.

## Technologies Used

- React.js
- Vite
- JavaScript (ES6)
- HTML5
- CSS3
- localStorage

## Features

- Create Draft
- View Drafts
- Edit Draft
- Delete Draft
- Save Drafts in localStorage
- Responsive UI
- Async Save Simulation

## Installation

```bash
npm install
npm run dev
```

## Expected Outcome

A React application that allows users to create, edit, delete, and store drafts using browser localStorage.
