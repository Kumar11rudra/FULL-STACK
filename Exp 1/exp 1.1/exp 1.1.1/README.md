# Experiment 1.1.1

## Title
Dynamic Social Media Post Composer

## Aim
To design and develop a dynamic post composer interface supporting multiple platforms with constraint validation.

## Objectives

- Understand multi-platform content handling.
- Implement real-time validation.
- Design responsive and reusable UI components.

## Technologies Used

- React.js
- Vite
- JavaScript (ES6)
- HTML5
- CSS3

## Features

- Multi-platform selection
- Character counter
- Character limit validation
- Media upload
- Responsive design
- Real-time validation

## Installation

```bash
npm install
npm run dev
```

## Expected Output

A responsive React application that allows users to compose social media posts with platform-specific validation and media upload support.
