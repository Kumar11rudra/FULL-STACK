# Experiment 1.2.1

## Title

Centralized State Management using Redux Toolkit

## Aim

To design and implement a centralized state management system using Redux Toolkit for managing posts and platform-related data.

## Objectives

- Understand global state management.
- Configure Redux Toolkit.
- Implement CRUD operations using slices.
- Connect React components with Redux Store.
- Build scalable frontend architecture.

## Technologies Used

- React.js
- Redux Toolkit
- React-Redux
- Vite
- JavaScript (ES6)
- HTML5
- CSS3

## Features

- Centralized Store
- Add Post
- Edit Post
- Delete Post
- Platform Selection
- Redux Slice
- React-Redux Hooks
- Responsive UI

## Installation

```bash
npm install
npm run dev
```

## Expected Outcome

A React application using Redux Toolkit to manage posts through a centralized global state.
