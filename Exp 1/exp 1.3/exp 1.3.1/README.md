# Experiment 1.3.1

## Title

JWT Authentication using React

## Aim

To implement a stateless authentication system using JSON Web Tokens (JWT).

## Technologies

- React.js
- Vite
- JavaScript
- jwt-decode
- LocalStorage

## Features

- Login Form
- Mock User Authentication
- JWT Token Simulation
- Token Storage
- Token Decoding
- Dashboard Access
- Logout Functionality

## Installation

```bash
npm install
npm run dev
```

## Learning Outcomes

- JWT Authentication
- Stateless Sessions
- Token Storage
- Token Validation
- Protected UI
