# Experiment 1.3.2

## Title

Role-Based Access Control (RBAC) using React Router

## Aim

To implement role-based access control and secure application routes based on user permissions.

## Technologies

- React.js
- React Router DOM
- Context API
- JavaScript
- CSS3
- Vite

## Features

- Login with Role Selection
- Admin, Editor and Viewer Roles
- Protected Routes
- Unauthorized Page
- Dynamic Navigation
- Conditional UI Rendering
- Logout

## Installation

```bash
npm install
npm run dev
```

## Learning Outcomes

- Authentication vs Authorization
- Role-Based Access Control
- Protected Routes
- Dynamic UI Rendering
- Secure Navigation
